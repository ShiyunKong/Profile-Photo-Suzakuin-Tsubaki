# Profile-Photo-Suzakuin-Tsubaki
Stable Diffusion created profile photo


设计简述：
设定为元宇宙里的赛博城市。
天上有核聚变太阳炉生成近乎无限的能源，同时有象征着赛博攻击的绿色月球。GPU算力和连接的朗基努斯信号发射器是力量的体现。（本来椿手里的双刀换成了双卡🤣）竖起来的耳朵就当做是信号接收器了✌
